== Description ==

Search Box Google par JM Créa permet d’intégrer le mini moteur de recherche de votre site dans les résultats Google.

Le plugin est paramétrable et le code javascript se place automatiquement entre les balises head.

== Screenshots ==

1. Paramètre personnalisé (Back-office)
2. Aperçu des résultats Google



== Installation ==

1. Uploadez `search-box—google-par-jm-crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Search Box Google par JM Créa’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite dans l’onglet ‘Outils -> Search Box Google’
